## ゼミ研究

手話は、聴覚または言語障害者が他者とコミュニケーションをするために使う言語です。 しかし、多くの人が手話を知らないことが、彼らの日常生活に大きく影響している。

本研究ではでは、手のキーポイントアルゴリズムにより、連続的な手のデータを抽出する。そして、動きによって分割される。最後に、分割されたデータを自然言語処理モデルに入力し、手話の翻訳を出力する。

この研究により、より迅速かつ正確な手話認識を実現することができる。聴覚または言語障害者の日常生活を便利にするとともに、手話の普及を促進することができます。
